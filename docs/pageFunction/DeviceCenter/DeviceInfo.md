# 水表管理系统

## 系统概览
![表计管理系统界面示意图](/设备中心/表计信息.png)

## 主要功能模块

### 基础信息管理
- **设备标识**：管理水表的名称、ID和型号信息
- **品牌信息**：记录水表的制造商和引擎类型

### 计量参数配置
- **类型设置**：区分不同种类的水表和计量能源类别
- **参数调整**：配置折算系数、倍率和启用底数
- **统计选项**：设置是否纳入能耗统计和测量统计

### 层级关系管理
- **表计编号**：管理水表的唯一编码
- **父子关系**：设置主表和子表的从属关系，实现分级计量

### 安装信息记录
- **位置管理**：详细记录水表的安装地址和具体位置
- **时间追踪**：记录安装日期，便于设备寿命和维护周期管理
- **供应信息**：管理水源供应点位和区域划分
- **用途分类**：记录设备的使用场景和服务对象
- **信号监测**：显示设备通信信号强度，确保数据传输稳定

### 备注信息
- **自定义说明**：添加设备相关的特殊情况说明
- **维护记录**：可记录设备的维修历史和特殊事件
